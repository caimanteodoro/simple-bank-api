# Simple Bank

Este é um projeto iniciante de um **sistema bancário simples** em Python.

## Funcionalidades

- Criar usuário
- Criar conta
- Depositar
- Sacar
- Ver saldo
- Ver histórico de transações

## Como rodar

1. Abra o terminal ou Repl: [https://replit.com/](https://replit.com/)
2. Copie os arquivos `main.py`, `user.py`, `account.py` e `transaction.py`
3. Execute `main.py`  
4. Siga o menu interativo para testar o banco
