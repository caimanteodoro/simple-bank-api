# Simple Bank

Este é um projeto iniciante de **sistema bancário em Python**, com menu interativo em linha de comando.  
Permite criar usuários, contas, fazer depósitos, saques e consultar saldo e histórico de transações.

## Funcionalidades

- Criar usuários
- Criar contas
- Depositar
- Sacar
- Ver saldo
- Ver histórico de transações
- sair
  
## Como rodar

1. Abra o projeto no Repl: (https://replit.com/@ccaimanteodoro/simple-bank-api)  
2. Clique em **Run**  
3. Siga o menu interativo no terminal para testar o banco.
